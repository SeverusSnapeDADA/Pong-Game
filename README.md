# Pong Game

## Documentation

### Features
- Classic Pong gameplay with simple mechanics.
- Two-player mode allows for competitive play.
- Score tracking to keep track of points.
- Adjustable difficulty levels.
- Sound effects and animations for enhanced experience.

### Controls
- **Player 1:**
  - Use the `W` key to move up.
  - Use the `S` key to move down.

- **Player 2:**
  - Use the `UP` arrow key to move up.
  - Use the `DOWN` arrow key to move down.

### How to Play
1. Start the game and choose a player.
2. Control your paddle using the designated keys.
3. Aim to hit the ball and prevent it from passing your side of the screen.
4. Score points by getting the ball past your opponent's paddle.
5. The player with the most points at the end of the game wins!

### Game Mechanics
- The ball will bounce off the paddles and the top and bottom edges of the screen.
- If the ball goes past a paddle, the opposing player scores a point.
- The speed of the ball increases over time to make the game more challenging.

2. Open the game directory.
